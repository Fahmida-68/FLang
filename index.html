Documentation

FLang is a localized procedural toy programming language ecosystem built natively on top of Python compiler design patterns.

General

shuru flang is the entrypoint for the program and all programs must end with shesh flang. Anything outside will be ignored.

This will be ignored

shuru flang
    // Write code here
shesh flang

This too

Variables

Variables can be declared dynamically anywhere inside the entry scope blocks using the keyword rakho.

shuru flang
    rakho a = 10
    rakho b = "two"
    rakho c = 15
    a = a + 1
    b = 21
shesh flang

Types

Numbers and strings are fully integrated. Conditional expressions resolve standard comparisons directly inside control definitions.

shuru flang
    rakho a = 10
    rakho b = 10 + (15 * 20)
    rakho c = "FLang Engine"
shesh flang

Built-ins

Use the default command standard keyword bol to print output values or statements directly onto the terminal logs.

shuru flang
    bol "Hello World"
    rakho a = 10
    bol a
shesh flang
